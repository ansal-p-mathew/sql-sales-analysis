# SQL Pizza Sales Analysis

## Project Overview


This project analyzes a pizza sales dataset using SQL to uncover key business insights such as top-selling pizzas, revenue trends, and customer ordering patterns.

## Tools Used

* SQL (MySQL / PostgreSQL)
* GitHub

## Dataset

The dataset includes:

* Orders
* Order Details
* Pizzas
* Pizza Types

## Objectives

* Analyze total revenue and orders
* Identify best-selling pizzas
* Understand sales trends over time
